## Purpose

BetterContact API purpose is to be the only API you need to integrate for your leads and contact enrichment. No more need to create in-house waterfall enrichment process with dozens of API calls to several providers.

### How it works

<AccordionGroup>
  <Accordion icon="upload" title="Submit your lead(s) to enrich">
    BetterContact API have a signle endpoint to enrich a single lead or a batch up to 100 contacts at a time.
    [>See endpoint](/api-reference/endpoint/create)
  </Accordion>
  <Accordion icon="download" title="Fetch results">
    Waterfall enrichment requires a couple of seconds to requests all providers and verify data. Once you have submited your leads, you have to fetch the enriched data. /
    You can also provide a webhook url in the submit call to instantly get the results without need to fetch it.
    [>See endpoint](/api-reference/endpoint/get)]
  </Accordion>
</AccordionGroup>

### What BetterContact does

<AccordionGroup>

<Accordion icon="1" title="Request providers in relevant sequence">
  According to the leads information provided, BetterContact determine the optimal sequence to find requested data.
</Accordion>
<Accordion icon="2" title="Verify provider data once found">
  Once a provider return a data, we verify it and deliver it to you only if valid.
</Accordion>
<Accordion icon="3" title="Request next providers untill data is verified">
  If not, BetterContact will request next providers untill correct data is found
</Accordion>

</AccordionGroup>
