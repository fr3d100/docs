title: "API Rate Limits"
description: ""
---

All API endpoints share a common rate limit. This limit applies across all endpoints collectively, not individually. When the rate limit is exceeded, the API will return a `429 Too Many Requests` status code.

<Note>
  API Rate Limit is 60 requests per minute per account (per API key). Would like to have higher API Rate Limits?  [**Contact us**](mailto:contact@bettercontact.rocks).
</Note>
