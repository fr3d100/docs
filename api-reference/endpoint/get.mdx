## Get Enrichment Results

Retrieve the enrichment results for a specific request ID.

### Request

`GET /async/{request_id}`

### Path Parameters

| Parameter   | Type   | Description                           |
| ----------- | ------ | ------------------------------------- |
| request_id  | string | The ID of the enrichment request      |

### Response

| Status Code | Description                          |
| ----------- | ------------------------------------ |
| 200         | Enrichment results retrieved         |
| 401         | Not authorized                       |
