When sending values to the [Lead Finder](/api-reference/endpoint/lead_finder_post) endpoint, several filters require **exact** values from a predefined list. Use the references below to look up accepted values.

<Note>
  Lookup is case-sensitive — `"Software Development"` and `"software development"` are not equivalent.
</Note>

These taxonomies are derived from the Landbase data provider. They may evolve over time — bookmark this page.

Used by `lead_function`. Functions accept the **same values as Departments above** — both filter on the same underlying field (Landbase `member_departments`). If you set both `lead_department` and `lead_function` in the same request, the values are merged.
